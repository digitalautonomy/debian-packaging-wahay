Downloads

Wahay is in an early stage of development. Not all of its functionalities have been developed and you may find errors! Download it at your own risk!

At the moment it's available only for Linux 64 bits. In the future we plan to provide binaries for other operating systems.

Linux downloads

This binary should be compatible with most of the recent/latest Linux distributions. In order for Wahay to work you will need to install Mumble and Tor (>=0.3.5).

This is a compressed file that bundles Wahay, Tor and Mumble. It is not necessary to install on the system or have root privileges, all you have to do is uncompress it and double-click on the Wahay app.

Different options to install Wahay are available. The Binary file is Wahay application without Mumble and Tor.
